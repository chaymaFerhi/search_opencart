## Test Plan – Cart Functionality

**Project Name:** OpenCart Manual Testing
**Module Name:** Cart
**Test Type:** Manual Testing

## Environment

* Browser: Google Chrome
* Operating System: Windows
* Application URL: https://www.opencart.com/index.php?route=marketplace/extension&member_token=f88a6f1ba895ebf5f0fa53a74cac4692

## Objective
To verify that the cart functionality works correctly when adding, updating, and removing products.

## Scope

* Add product to cart
* Add multiple products
* Update quantity
* Remove product
* Empty cart
* Verify cart total calculation

## Out of Scope

* Payment processing
* Checkout functionality

## Test Approach
Manual testing will be performed by executing cart-related test cases and verifying expected behavior.


## Deliverables

* Test Cases
* Bug Report
* screenshots
* Test Summary
